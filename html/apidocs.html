Capture Screenshot

POST

The main endpoint for generating high-quality screenshots. It supports real-time rendering, full-page capture, and retina resolution across all modern browsers.

https://api.screencapr.com/v1/capture

Query Parameters

Parameter Type Description
url string REQUIRED The fully qualified URL of the page you want to capture.
viewport string The width and height of the browser window. Format: 1920x1080.
full_page boolean Scroll the entire page and capture the full height. Defaults to false.
format enum Output format. One of: png, jpeg, pdf.

Response Example

A successful request returns a JSON object with the screenshot URL and metadata.

{
  "id": "sc_9f823jks",
  "url": "https://example.com",
  "screenshot_url": "https://cdn.screencapr.com/i/abc.png",
  "created_at": "2023-11-10T14:48:00Z",
  "status": "completed"
}

Request Example
1 curl -X POST https://api.screencapr.com/v1/capture \
2 -H "Authorization: Bearer YOUR_API_KEY" \
3 -H "Content-Type: application/json" \
4 -d '{
5 "url": "https://screencapr.com",
6 "viewport": "1280x800",
7 "full_page": true
8 }'
